# Logistic Map
This code plots a bifurcation diagram of logistic map described below.
```math
x_{n+1} = rx_n(1-x_n)
```
This is one of the famous examples of chaos and fractal. [^1] The codes are provided in Python and R.
## Requirements
```
numpy, matplotlib
```
